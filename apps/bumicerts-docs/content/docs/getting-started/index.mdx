This section walks you through the prerequisites, installation, and a
quickstart tutorial so you can have the full Bumicerts stack running
locally.

## Prerequisites

- [Bun](https://bun.sh) v1.1 or later
- [Node.js](https://nodejs.org) v22 or later (required by some tooling)
- [Docker](https://www.docker.com/) (for the indexer's PostgreSQL database)
- A GitHub account (for cloning the repository)

## Sections

- **[Installation](/docs/getting-started/installation)** — clone the repo,
  install dependencies, and configure environment variables.
- **[Quickstart](/docs/getting-started/quickstart)** — run the dev servers and
  create your first record.
